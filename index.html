Why Web Development??

Web development is a flexible way to enter the tech industry. Web developers can work as freelancers, join a team or start their own business.

Web Design

Creating graphics, page layouts, website structure & navigation. Web designers also need to ensure that the website's navigation is fast

Frontend development

Using design tools, frameworks, libraries and coding to build websites that are easy for visitors to interact with

Responsive design

Ensuring that HTML, CSS and JavaScript elements like menus, text & buttons are clear & usable across all devices. This helps to ensure consistent content delivery
